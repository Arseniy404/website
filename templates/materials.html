Interview Questions
Logistic Regression
  • Why does the loss function have this form? Can it be replaced with MSE?
  • Why log-likelihood? (numerical stability, simpler gradient, SGD over samples)
  • Can sigmoid be replaced? What properties does the activation need?
  • Do we converge to the same solution from different initializations? (convexity)
  • Should features be normalized, and what does it affect?
Tree Ensembles
  • How to choose tree depth for boosting vs RF, and why differently?
  • What is bias and variance in the error decomposition?
  • What should a tree output for boosting to solve classification?
  • How to estimate feature importance with RF?
  • Can you overfit by adding more trees? (RF — barely, boosting — yes)
Overfitting and Regularization
  • Why does regularization help with overfitting?
  • Why does L1 sparsify weights? (diamond geometry / subgradient at zero)

Metrics
  • What's wrong with accuracy and why do we need so many metrics?
  • Can we maximize only precision (or recall) while ignoring the other?
  • Why is F1 the harmonic mean rather than arithmetic?
  • Which is better under class imbalance — PR-AUC or ROC-AUC?
  • What does the PR curve look like for a random classifier?
  • Why is ROC-AUC invariant to class balance while PR-AUC is not?
Statistical Significance
  • Can you conclude from one test set that model A is better than model B?
  • How to quickly estimate significance of metric differences? (statistical test / bootstrap)
  • How to handle multiple comparisons? Why does the test set "wear out"?
